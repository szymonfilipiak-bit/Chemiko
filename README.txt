CHEMIKO — wersja do publikacji

Po założeniu konta Vercel:
1. Utwórz konto GitHub.
2. Utwórz nowe publiczne repozytorium, np. chemiko-strona.
3. Wgraj wszystkie pliki z tego folderu: index.html, vercel.json, README.txt i folder assets.
4. Na Vercel wybierz Add New > Project.
5. Połącz repozytorium i kliknij Deploy.
